GitHub'a yükleme adımları
=========================

1) GitHub.com'da yeni bir REPO oluşturun (Public, README eklemeyin).

2) Aşağıdaki komutları sırayla çalıştırın.
   REPO_URL kısmını kendi repo adresinizle değiştirin.
   Örnek: https://github.com/KullaniciAdi/oyun-koleksiyonu.git

   cd "C:\Users\PC\Desktop\REACT PROJE"

   git remote add origin REPO_URL

   git push -u origin main

3) İlk push'ta GitHub girişi istenebilir (tarayıcı veya token).

REPO_URL = GitHub'da "Create repository" sonrası çıkan adres (HTTPS veya SSH).
